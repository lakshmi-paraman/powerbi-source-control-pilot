# Power BI Source Control Pilot

## Purpose

This repository stores Power BI PBIP artifacts for source control readiness.

## Scope

This pilot includes:

- One Power BI report exported as PBIP
- One Power BI semantic model exported as PBIP
- Repository structure

- Restore/runbook steps
